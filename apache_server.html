How to setup an Apache Server: the simplest config

  • On the server, install the httpd package.
  • In /etc/httpd/conf, modify the httpd.conf file, by means of specifying the ServerName directive, e.g:
    ServerName www.example.com .
  • Put the file relative to the website in /var/www/html, that will be the default directory in which Apache will be searching (it can be modified specifying the path in the DocumentRoot directive in the httpd.conf file).
  • Configure /etc/hosts to access the website by name.
  • Start and enable httpd service.
  • Check if website can be displayed from the server itself (it can be useful to use elinks, just for a quick try).
  • Open port 80 (Hint: firewall-cmd --add-port=80/tcp, firewall-cmd --add-port=80/udp or just firewall-cmd --add-service=http).
